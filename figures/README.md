# Figures

These are the computational figures generated by the scripts in `code/`, at
300 DPI with dual PSI/MPa units. Filenames reflect the order in which the figures
were generated and **do not** correspond to the figure numbers in the paper, which
follow the paper's own sequence. Each figure here is reproducible by running the
associated script.

| File | Content | Generated by |
|---|---|---|
| `fig1_stress_vs_yield.png` | Operating stress vs. derived yield strength per loading mode | (FEA plotting) |
| `fig3_safety_factor.png` | Safety factor by loading mode | (FEA plotting) |
| `fig4_displacement.png` | Maximum displacement by loading mode | (FEA plotting) |
| `fig5_physiological_loads_bones.png` | Equivalent physiological loads across bones/activities | (FEA plotting) |
| `fig6_fatigue_SN_curve.png` | Traditional S–N (stress-life) fatigue curve | `pcl_fatigue_analysis.py` |
| `fig6b_fatigue_endurance_margin.png` | Operating stress vs. endurance limit (threshold/margin view) | `pcl_fatigue_analysis.py` |
| `fig7_cyclic_waveform.png` | Zero-based (R=0) cyclic loading waveform | (fatigue plotting) |
| `fig8_degradation_model.png` | PCL strength + molecular-weight decline over time | `pcl_degradation_model.py` |
| `fig9a_mesh_vonmises.png` | Mesh convergence — Von Mises stress | (mesh plotting) |
| `fig9b_mesh_safetyfactor.png` | Mesh convergence — safety factor | (mesh plotting) |
| `fig9c_mesh_displacement.png` | Mesh convergence — displacement | (mesh plotting) |

Note: the fatigue endurance limit shown in the S–N and margin figures is estimated
as 0.4 × ultimate tensile strength (a conventional approximation, not measured PCL
data), as documented in `../DATA_DICTIONARY.md` and in the script headers.
